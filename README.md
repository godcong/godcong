## 👋 Hello, I'm GodCong

<table>
<tr>
<td width="62%" valign="top">

### 🚀 Founder of [OrigAdmin](https://github.com/origadmin) · Go Framework Architect

> *"Building frameworks so others can build faster."*

I'm the creator and maintainer of **[OrigAdmin](https://github.com/origadmin)** — an open-source Go admin framework ecosystem. My days are spent architecting **modular backend services**, designing **admin UI systems**, and crafting developer tools that make building production-ready applications a breeze.

<br/>

**🔭 Currently focused on**
- 🏗️ [**OrigAdmin**](https://github.com/origadmin) — Go 全栈管理框架生态
  - [**runtime**](https://github.com/origadmin/runtime) — ⭐ 基于 go-kratos 的运行时核心框架（配置/服务发现/中间件/存储抽象）
  - [**admin**](https://github.com/origadmin/admin) — 🏢 后端框架（基于 runtime 构建）
  - [**contrib**](https://github.com/origadmin/contrib) — 🔌 基于 runtime 的扩展贡献库
  - [**toolkits**](https://github.com/origadmin/toolkits) — 🧰 基础工具链
  - [**webui**](https://github.com/origadmin/webui) / [**shadcn-admin-design**](https://github.com/origadmin/shadcn-admin-design) — 🎨 管理后台前端 (ShadcnUI)
  - [**adptool**](https://github.com/origadmin/adptool) / [**abgen**](https://github.com/origadmin/abgen) — ⚡ 适配/代码生成工具

**🌱 Also building**
- [**fate**](https://github.com/babyname/fate) — 2.4k ⭐ 现代科学起名引擎（八字/五行/周易）
- [**chronos**](https://github.com/godcong/chronos) — 中国历法/干支/节气库
- AI tooling & cloud-native tooling

**💡 Open source believer** — if it solves a real problem, it deserves to be shared. Let's build together.
